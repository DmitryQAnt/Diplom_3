### Yandex Practikum's Diplom part 3
**Description**: In my autotest study project I propose the test coverage of some functionality of the StellarBurger UI.

**Used technologies**: Java 11, Selenium-Java 4.9.1, JUnit 4.13.1, Maven 3.9.0, Maven surefire-plugin 2.22.2, Allure-maven 2.10.0, Rest-assured 5.3.2.

Run tests for Chrome Browser:`mvn test -Dbrowser`

Run test for Yandex Browser:`mvn test -Dbrowser=yandex`
